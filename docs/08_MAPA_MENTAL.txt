┌─────────────────────────────────────────────────────────────────────────────┐
│                       FLUJO DE TRABAJO TERRAFORM                            │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  1. terraform init                                                          │
│     └─→ Descarga providers y módulos                                       │
│         └─→ Resultado: .terraform/ creado, .lock.hcl generado              │
│                                                                             │
│  2. terraform plan                                                          │
│     └─→ Lee código y variables                                             │
│     └─→ Conecta a AWS                                                      │
│     └─→ Calcula qué crear/cambiar/eliminar                                │
│         └─→ Resultado: tfplan generado (preview)                           │
│                                                                             │
│  3. terraform apply                                                         │
│     └─→ Ejecuta el plan                                                    │
│     └─→ Crea 30 recursos en AWS                                            │
│     └─→ Actualiza state en S3                                              │
│         └─→ Resultado: Infraestructura completa en AWS                     │
│                                                                             │
│  4. terraform output                                                        │
│     └─→ Muestra outputs                                                    │
│         └─→ Resultado:                                                     │
│             - IP privada del DC                                            │
│             - IP pública del cliente Windows                               │
│             - IP pública del cliente Linux                                 │
│             - Comandos RDP/SSH listos para usar                            │
│                                                                             │
│  5. terraform destroy (cuando termines)                                     │
│     └─→ Elimina TODOS los recursos                                         │
│         └─→ Resultado: Evita costos, infraestructura eliminada             │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
